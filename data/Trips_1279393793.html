Open the Trips screen using the  symbol in the cadis control panel:

With the help from the Trips screen you have complete control over your last mile routes.

You have an overview of your routes with

  • Stop list in table form and graphical form on the map

  • Tour status

  • Driven route and remaining route on the map

  • PTA (Planned Time of arrival) and ETA (Estimated Time of Arrival) at stops and depots.

In addition you can schedule and reschedule shipments in the trips screen. You can do this both in the tabular view and in the map view.

Filter area

In the filter area you can define which routes can be seen in the other areas.

List of routes

The selected routes are displayed in the route table.

You can see the following information

  • Tour status and vehicle utilization with of collections and deliveries

  • Planned and estimated times

  • Vehicle data, driver, freight carrier

  • Driving distance and time

By highlighting one or more lines (selecting with the mouse) these routes are displayed on the map.

Context menu

Right-click on one or more routes to perform the following actions. If an entry is deactivated (grey), the reason for this is displayed as a tooltip.

  • Route details, Route details#2, Route details#3
    This opens the Route details screen. You can open a total of 3 windows with route details in parallel. By selecting the menu item you can choose which window the route is displayed.

  • Assign vehicle
    Assign a vehicle to a route that have not been approved for loading

  • Change the vehicle
    You assign another vehicle to the route.

  • Release for the same vehicle
    If the driver has refused the route, you can dispatch it on the same vehicle again.

  • Send message
    Send a text message to the assigned vehicle

  • Optimize Sequence
    Switches to the Sequence optimization screen

  • Cancel dispatching
    Cancels the dispatching for all shipments on this route

  • Delete Routes
    Deletes a route if it contains no shipments

  • Release Loading
    Releases the route for loading, Registration for loading by a mobile device is then available

  • Deny loading completion
    Denies loading end reported by the driver

  • Close loading
    Finalize completion of loading after the driver has reported the end of loading

  • Print
    Print diverse documents

  • Fit route in map
    A suitable map section is selected so that all tours are visible

List of stops

You can see the stops in the lower table on the Stops tab.

Context menu

Using the context menu you can run the following actions

  • Shipment details
    The Shipment details screen is displayed. If a stop contains several shipments, you can scroll between the shipments of the stop in the shipment details.

  • Dispatch cancellation
    You can cancel the dispatching of unprocessed collections.

  • Dispatch again
    If the driver has refused the collection you can dispatch it on the same route again.

  • Stop details
    In the Stop details screen you can see the details of the stop. You can enter or change the collection or delivery status manually.

List of shipments to be dispatched

Shows the list of shipments to be dispatched in the "To be dispatched" tab. The table corresponds exactly to the shipment overview in terms of content and operating option. You can display the filter block using the  symbol and hide it again using the symbol, to leave more space for the table.

Dispatching process

Dispatching is available from this overview by using the drag & drop function. To do this, select one or more shipments and drag them with the mouse to a route in the route table or to a connecting line between 2 stops on the map.

When dispatching on the route table, the shipments will be added to the end of the list, unless they match an existing stop.

When dispatching on the map, the shipments are inserted between the stops on whose connecting line the shipments were dragged onto.

If the Dispatching tab is not large and clear enough, you can also dispatch from the List of shipments screen with drag and drop as described above.

Map area

Map options

  • To be dispatched
    You can select which shipments should be displayed on the map:
    – all collections and/or shipments to be scheduled.
    – those that were filtered into tabs to be dispatched.

  • Remaining and driven route shows the actual route driven based on GPS data and the planned remaining route

  • Show traffic info: shows traffic jam information, road closures, construction sites etc.

Operating the map window

  • Slider / + / – at the right edge of the map, or mouse wheel:
    Use this to zoom in on the map

  • Function bar at the top left of the map window 
    With the  symbol you change the function of the left mouse button when a range is marked while holding down the mouse button

  • the marked area is zoomed in

  • all collections, shipments and stops are marked in order to process them together, for example to schedule them together

With the  symbol you select the initial map section or scroll back and forth between the previously selected map sections.

  • Hold down the right mouse button and move the selected map section

  • Holding down the left mouse button, you can plan or re-dispatch by moving one or more selected objects (stops, collections, shipments) to the connecting line between 2 stops. Collections and shipments are scheduled in this way, stops are re-dispatched.
    If you move a stop to another position within the same route, you change the order of the stops.

Menu bar and footer

Regular collections / routes menu

  • Manage recurring collections: You will be directed to the corresponding screen for creating, changing and deleting recurring collections.

  • Generate recurring collection orders
    Recurring collections are created every night for the next day. If you want to create recurring collections beforehand, select the required day and choose OK.

  • Manage recurring trips for last mile / line haul
    You will be directed to the corresponding screen for creating, changing and deleting recurring trips last mile / line haul

  • Create recurring trips for last mile / line haulRecurring trips created every night for the next day. If you want to create recurring trips beforehand, select the required date range and choose Create trips

Display menu

Here you can define in which cases the map section should be zoomed automatically.

You can also choose a horizontal layout in which the two table areas are displayed next to each other.

Buttons in the footer

  • Shipments to be dispatched
    The List of shipments is opened with the Schedule master search filter set, filter setting: secondary distribution transport and direct shipments to be scheduled.

  • Automated dispatching
    An automated dispatch is run for all scheduled shipments and collections.
    If you only want to perform an automated dispatch on some of the shipments, select the desired shipments in the List of shipments and choose Automated dispatching in the context menu

  • New
    Here you create a new secondary distribution route.
