Creation

Use the New button in the shipment overview or the :AuftragsUebersicht: button in the cadis control panel.

Shipment Number

The shipment number is automatically generated by cadis when you save.

Optionally you can enter a tracking number which you can search for later in the shipment overview.

Addresses

You can select the addresses from master data or enter them yourself.

The customer master data is used to select a consignor, while the shipper and recipient can be selected from the stop location master data.

If, for example, you select the collection address from the master data, all other collection related data, such as collection times, handling instructions and service times are transferred in addition to the actual address data.

Click on the :Suche: symbol to open the master data for selecting an address.

If you have selected the address from the master data, the address is permanently linked to the master data. The address fields are grayed out and cannot be modified or amended.

You can use the :Redo: symbol to remove the link to the master data and you can then modify the address fields. Once you have removed the link, collections can no longer be assigned to the originally selected customer or stop location in later changes.

Collection Date/ Collection Time

The current date is set by default. You can also specify a later date or specify a date range for the collection.

The collection times are taken from the master data according to the day of the week.

Shipment Quantity

The entry of a weight is mandatory. If you enter individual shipment items with weights, the calculated total from the shipment items is taken over automatically as the total weight.

Additional Shipment Characteristics

The 3 checkboxes under the shipper address applies to the collection and the one under the consignee address applied to the delivery. An explanation of the symbols (tail lift, crane, additional staff) is displayed as a tooltip when you hover over it with the mouse.

Shipment Positions

You can enter more detailed information about the collection. You can add a new entry by clicking on the  symbol and delete the marked entry with :PosAdd:.

Handling Instructions

In the Handling instructions tab you can add a note text with :PosAdd: and delete the selected text. Select "Collection” as the type if the text is to be displayed on the mobile device when you collect it. The "Internal" type is only displayed in cadisCONTROL. Handling instructions with the type "General" are displayed for all activities.
