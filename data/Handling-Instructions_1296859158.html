CADIS Knowledge Base : Handling Instructions

Created by Julian Beyer, last modified on Sep 04, 2025

You can reach the screen for creating handling instructions by using the master data overview (:regeldefinition: symbol in the cadis control panel).

Handling instructions provide information on how to deal with packages in the transportation process, e.g. when loading or at delivery.

The fields marked with an asterisk are mandatory.

  • ID: Code of the handling instruction that is referenced per shipment in the cadis data interface (XML element: HandlingInstruction).
    Alternative: Any ID for instructions that are NOT referenced in the cadis data interface.
    Note: Any handling instruction can be added to a shipment in cadis via workflow rules and stop locations.

  • Text: Description shown to users (e.g. “Deliver to neighbor”)

  • Important: If set, a pop-up with this information shows up when the driver starts the delivery/collection stop.

  • Trip relevant: If set, this instruction is shown in the trip instructions.
    (see https://cadissoftware.atlassian.net/wiki/x/E4BNTQ)

  • Display order: Most frequently used instructions can be shown at the top of a selection list by setting the display order accordingly.
