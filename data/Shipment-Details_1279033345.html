You can reach the Shipment details overview showing the shipment details by double-clicking a shipment or using the context menu in the shipment overview and from the list of shipments in the Loading, Unloading and Inventory processes.

You can see the most important shipment data in the header section of the screen and you can modify parts of the shipment data. The different tabs show the details of the shipment and its processing status.

Menu

  • Edit
    Standard functions for the window layout

  • Print
    You can print the dangerous goods annex, delivery notes, collection order, temperature report, shipment details

Toolbar

  • :refresh: Refreshes the screen (shipment data)

  • You can manually geocode the collection and delivery addresses if they were not geocoded automatically.

  • Cancelled values:By ticking this box, cancelled lines are displayed in the tables.

  • ,  Previous / Next shipment. You can use these buttons to scroll between the shipments if you have selected several shipments, e.g. by multiple selection in the shipment overview using the context menu.

Tabs

The green symbol means that the tab contains information. The white symbol  means that the tab is empty.

Positions Tab

Shows the different shipment positions.

Instructions Tab

Shows instructions. You can enter instructions by using the :PosAdd: symbol or delete the instructions that are recorded in cadis using the :PosSub: symbol.

Notes are either assigned to a process or more generally to all processes and are thus displayed in all of them.

Details Tab

Shows further information about the shipment

Packages Tab

You can see the package information and the package status.

Money amounts Tab

Shows the cash on delivery and freight amounts.

Loading equipment Tab

All loading equipment is listed here. These are those mentioned under shipment items and those that may have been added as loading aids.

Dangerous goods Tab

Displays the detailed dangerous goods data.

Dates Tab

Displays the dates e.g. collection and delivery dates.

Addresses Tab

Displays all addresses in relation to the shipment. The most important addresses (client, consignor, consignee) are also displayed in the header of the shipment.

Contacts Tab

Shows contacts

References Tab

Shows reference numbers e.g. order numbers

Loading locks Tab

Displays the loading locks that are set by a host system or manually in cadis.

Status

Shows the detailed status information relating to the shipment

Photos Tab

Displays all photos added to the shipment or its packages in all processes

Weights / Volumes Tab

In the Type of reference column you can see if the weight and volume data refer to shipment items or to individual packages.

Customs Tab

Shows customs information

History Tab

Displays detailed shipment history

Notes Tab

Notes are a way to communicate between the dispatcher, customer service or user in an ordering depot.

Press the NEW button to create a new note:

and by ticking the corresponding checkbox, set which group of people the note must be actioned by (in the shipment overview, shown in the columns !D, !S, !N).

Double-click on a note to mark it as read

and by clicking the Done button it is marked as done.

A note without a recipient is immediately completed.

PSts / Processing status Tab

You can enter or change a processing status. The processing status indicates that a process is taking place or has taken place in some way outside of cadis.

Value added services Tab

Displays the value added services to be recorded during the delivery of the shipment.
