Sie erreichen die Extras über das Icon  in der cadis-Bedienleiste.

Unter diesem Menü sind verschiedene Funktion zur Administration und Einstellungen zusammengefasst.

Perspektiven

: Hiermit speichern Sie die aktuellen Einstellungen aller geöffneten Fenster (Größe, Position, angezeigte Spalten, deren Größe und Reihenfolge) unter einem Namen ab, unter dem Sie diese Einstellungen jederzeit wiederherstellen können.

: Hiermit wird die aktuelle Einstellung wieder gelöscht.

Benutzereinstellungen

Persönliche Einstellungen. Die Einstellmöglichkeiten sind in der Maske erklärt.

Ummelden

Ermöglicht das Neuanmelden an cadis mit anderem User, an einem anderen Depot mit einer anderen Sprache ohne cadis zu verlassen und neu zu starten.

Passwort ändern

Benutzerverwaltung

Anlegen, Löschen, Ändern von Benutzern und Rollen

Diese Maske zeigt die Liste der Benutzer.

Zum Neu Anlegen oder Ändern eines Benutzers erscheint die Maske Benutzer:

Ist Passwort ändern aktiviert, dann muss der Benutzer mit der ersten Anmeldung das Passwort ändern.

Sie weisen jedem Benutzer eine oder mehrere Rollen zu.

Lizenzverwaltung

Öffnet den Lizenzmanager im Bereich "Administration" in cadisSTUDIO.

Regelabholaufträge erzeugen

Sie können hier Regelabholaufträge für mehrere Tage im Voraus anlegen.

Mobilgeräte

Die Maske zeigt eine Übersicht über die eingesetzten Mobilgeräte.

Sie sehen den Zeitpunkt der letzten Anmeldung und das Fahrzeug, für das das Mobilgerät aktuell angemeldet ist.

Bei Selektion eines Eintrags können Sie von einem Mobilgerät eine Logdatei anfordern. Die Anforderung sehen Sie im rechten Bereich des Fensters. Dort sehen Sie, wenn die Logdatei übertragen ist und Sie Zugriff auf die Logdateien haben.

Die Mobilgeräte-ID ist nicht die Seriennummer des Gerätes, sondern eine durch die App generierte Zufallszahl.

Mobile Freigaben

Hier sind alle mobilen Freigaben eingetragen, die Sie erteilt haben.

Neue mobile Freigaben für Fahrer (Driver), die fest einem Fahrzeug zugeordnet sind, erteilen Sie in der Fahrzeugübersicht.

Neue mobile Freigaben für ein oder mehrere Fahrzeuge, einen Scanner oder einen Fahrer, die nicht fest einem Fahrzeug zugeordnet sind (MobileUser) erteilen Sie hier (Button Neu).

Sie können per Selektion alle mobile Freigaben bearbeiten oder die Freigabe entziehen (löschen).

Mobilgeräte Software

Hier verwalten Sie die Softwareversionen für die Mobilgeräte.

Mit Neu wird eine neue Softwareversion auf den Server geladen. Sie wählen in diesem Dialog die Zip-Datei mit der Mobilsoftware aus.

Anschließend können Sie die hochgeladenen Softwareversionen zuweisen, und zwar:

  • An alle Mobilgeräte aller Depots

  • An alle Mobilgeräte eines Depots

  • An einzelne Fahrzeug- bzw. Hallenscanner

Die entsprechenden Mobilgeräte machen dann beim Anmelden darauf aufmerksam, dass eine neue Software-Version zur Installation vorhanden ist. Solange die Version nicht als verpflichtend deklariert wird, kann der Mitarbeiter die Installation aufschieben.
