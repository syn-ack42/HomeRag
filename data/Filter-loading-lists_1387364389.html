CADIS Knowledge Base : Filter loading lists

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Loading lists can be filtered by the following properties in "My loadings":

  • Date

  • Traffic type

  • Sort loadings

The following options for "Traffic type" are available:

  • All

  • Last Mile

  • Line Haul

The following options for "Sort loadings" are available:

  • State

  • Gate

  • Departure time

The loading lists are loaded and displayed corresponding to the set filters.
