Open the Stop Location Master Data screen by using the master data overview (:regeldefinition: symbol in the cadis control panel).

Stop location master data is managed in cadis to maintain frequently visited stops with planning-relevant data.

Collection stop locations can be used collection order entry or for recurring collection templates.

Stop location overview

Select here which stop locations should be displayed.

Use the Create button, Open button or double click on a stop location to get to the Stop location details screen.

Stop location details

The stop location address and the position of the stop location on the map is shown in the Stop location data tab.

Further details are shown in individual tabs.

Set the appropriate check mark if the stop location is used as a collection address or a delivery address.

Processing times

These are used to calculate the time required for collection/delivery during route optimization, route sequence optimization and ETA calculation. It consists of a fixed part per stop and a variable based on the number of packages.

Vehicle attributes

These are used during the route optimization to assign the stop to the appropriate vehicle.

Opening hours

This defines which days and times collections/deliveries can and can’t take place. This data is used if nothing else is specified in the shipment.

Handling instructions

Provides the option of adding a note to shipments and collections that are delivered or collected at this stop.

Currently this is only checked on collections created manually using the cadis user interface.
