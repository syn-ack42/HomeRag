CADIS Knowledge Base : Send Message

Created by Julian Beyer, last modified on Aug 20, 2025

Open the Send Message screen using the :nachrichtSenden: symbol in the cadis control panel or using the context menu in the List of Vehicles screen or List of messages screen.

You can choose between sending to

  • One Vehicle

  • Multiple Vehicles

  • Vehicle group
