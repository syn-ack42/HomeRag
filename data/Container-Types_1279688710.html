CADIS Knowledge Base : Container Types

Created by Julian Beyer, last modified on Aug 21, 2025

You can reach the screen Container types and Swap bodies/Containers master data using the master data overview (:regeldefinition:  symbol in the cadis control panel).

First define container types before you define containers or swap bodies, because you must enter the container type for each container.

In cadis, a container is defined as everything that has a loading capacity but no wheels, i.e. not a vehicle.

Container types define common properties for containers / swap bodies, especially dimensions and weights.

Swap bodies / Container types

Swap bodies / Containers

In the SB / Container type block, enter only the Swap body / Container type. All other fields are taken from the container type master data and are only displayed here.
