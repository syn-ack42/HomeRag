CADIS Knowledge Base : Video: How to set up your org tree and branches in cadis

Created by Julian Beyer, last modified on Oct 16, 2025

This feature is available starting 2025.5 or on Free Trial systems.

This cadis tutorial guides users through setting up their first depot by creating an organizational unit in the Org Tree via the Master data dashboard. It covers essential setup steps including depot type selection, time zone and working days configuration, address and geolocation input, and activation for cadisCONTROL and cadisApp usage — laying the foundation for all future operations.

Transcript

Welcome! In this video, we’ll show you how to set up your first depot in cadis. This is the foundation you need before you can start working with the system.

To get started, open the Master data dashboard. Then go to the Org tree section. This is where you create your organizational structure – your depots, regions, or corporate levels.

You’ll notice a predefined depot called Depot1. This is required for your initial login. Without a depot, you can’t even access cadis. But Depot1 is just a starting point.

Once you're in, take it as a template and create your own depot based on your company’s actual setup. To create a depot, klick the “Create” button. We only need to fill in a few essential fields.

In the General Information section on the left, make sure to:

  • Select the type as Depot.

  • Choose the parent unit that this depot belongs to. For example, your company’s corporate unit. But you can also choose the predefined “SaaS” entry.

  • Set the default report locale, which defines the default language for reports related to this unit. For example, de for German.

In the Branch Details section, we enter the following:

  • The name of the depot or branch.

  • The time zone, so the system knows which time applies to loading, unloading, and tour planning.

Optionally, you can assign a logo for this branch.

Also, define the working days, for example, Monday through Friday. These days indicate when tours can start or end at this depot.

Make sure that the cadis checkbox is selected. This activates the unit for use in cadis.

Next, let’s move to the Branch Address. This part is especially important because it helps ensure correct routing and geocoding.

Enter the street, postal code, and city. We also strongly recommend filling in the latitude and longitude manually. Sometimes, external routing services like the PTV server might not recognize the full address correctly. Providing exact coordinates ensures accurate navigation.

The other sections like Branch Contact, Company Address, and PO Box can remain empty for now unless you want to add specific information for documentation purposes.

The display order is optional as well. But if you want to sort units in dropdown menus, you can assign a number here. A lower number means a higher position in the list.

Once all required fields are filled in, you can save the org unit. And it’s ready to be used in cadisCONTROL and the cadisAPP.

That’s it! You’ve successfully created your new depot.

Thanks for watching.
