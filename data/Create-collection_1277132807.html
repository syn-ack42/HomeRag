Creation

You reach the side for creating collections in the section Order Management / Collections / Button "Create" in cadisSTUDIO.

The fields marked with an asterisk are the mandatory for creating an unqualified collection order for which one or more shipments are to be collected without specifying the exact shipment quantity and recipient.

Shipment Number

The shipment number is automatically generated by cadis when you save.

Addresses

You can select the addresses from master data or enter them yourself.

The customer master data is used to select a consignor, while the shipper and recipient can be selected from the stop location master data.

If, for example, you select the collection address from the master data, all other collection related data, such as collection times, handling instructions and service times are transferred in addition to the actual address data.

Click on the  symbol to activate the master data search for selecting an address (only the field "Name" is active). Type in at least three characters and the matching master data entries are displayed. To confirm the entry, it has to be selected in the list.

If you have selected the address from the master data, the address is permanently linked to the master data. The address fields are grayed out and cannot be modified or amended.

You can use the  symbol to avtivate the manual entry or to remove the link to the master data to then modify the address fields. Once you have removed the link, collections can no longer be assigned to the originally selected customer or stop location in later changes.

Collection Date/ Collection Time

It is mandatory to enter a date, that is not in the past. You can also specify a date range for the collection.

The collection times are taken from the master data according to the day of the week.

Shipment Quantity

The entry of a weight is mandatory. If you enter individual shipment items with weights, the calculated total from the shipment items is displayed as "Calculated weight" in the shipment data. If no weight has been given for the shipment, the calculated weight is taken over as the total weight.

This logic also applies to the volume.

Additional Shipment Characteristics

There are additional shipment characteristics for the collection respectively the delivery. The three corresponding checkboxes and the respective explanation (tail lift, crane, additional staff) can be found unter the address data.

Shipment Positions

You can enter more detailed information about the collection. A new position can be created with the button "Create".

If all positions have the attribute "stackable", the shipment gets the attribute "stackable".

If at least one position has the attribute "Dangerous goods", the shipment gets the attribute "Dangerous goods".

In both cases – stackable and dangerous goods – the attribute can be overwritten manually in the shipment data.

Handling Instructions

You can enter a handling instruction with the button "Create".

Select "Collection” as the type if the text is to be displayed on the mobile device when you collect it. The "Internal" type is only displayed in cadisSTUDIO. Handling instructions with the type "General" are displayed for all activities.

Dispatch

After having enterred all relevant data you have the possibility to save and dispatch by one click, if the planned trip is already known and created in cadis.
