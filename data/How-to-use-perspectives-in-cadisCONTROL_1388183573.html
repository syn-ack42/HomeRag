CADIS Knowledge Base : How to use perspectives in cadisCONTROL

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

A Perspective is an individual window layout that you can set by clicking a named button. They can be used to configure different use cases and it a allows a very flexible handling of cadisCONTROL.

You can create more than one perspective.

The possibility to maintain (create and delete) perspectives can be found in the main menu bar under "Extras".

At the beginning you find yourself in the basic perspective, which has no name.

If there is created a perspective, there are no views available in any window. It is necessary to create these views.

After having done some changes on the views, the perspective must be saved again as well.

Defined views in windows always belong to one certain perspective.

Global perspectives can be exported and imported again via user preferences. Private perspectives cannot be exported, but they are saved user specific in cadis.
