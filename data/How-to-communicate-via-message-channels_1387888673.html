cadisAPP on Android requires Chrome version 71 (or Android the corresponding WebView) to visualize the Message Channels.

In a channel you can create text messages and include an image or photo (not on "Vehicle" channels), read messages, mark one together will all older ones as read. You will receive notifications about new messages.

Please note that cadis message channels are not "user to user" communication like WhatsApp & Co.. The purpose of cadis message channels is to support groups of users to transparently communicate about transportation related activities. There is no assumption of privacy, all channels are accessible by multiple users (e.g. all supervisors will see a "User – Supervisor" Channel)

In the following, you find details about the different message channel types.
Please refer to the user manual for the "Vehicle" channel documentation as it is established functionality.

Supervisors and Users

Supervisors: Your own channel

If you have logged in via user account, you always have access to a "Supervisors" channel. Only you and the group of supervisors have access. You can reach the supervisors. Each of the supervisors can reach you.

This is true for supervisors as well.

If you have logged in cadisAPP via barcode scan (and possible entering a pin but not your user credentials), you don’t see the Supervisors channel, because without a user account login, cadis do not know which user you are.

Users: The supervisors’ view on all users’ channels

This chapter is relevant for supervisor only.

If you have logged in in cadisSTUDIO with a supervisor permission, you have access to all users’ "Supervisors" channels.
You see each such channel named after the user.

(If you have logged in via cadisAPP, you won’t see the user channels and will not receive notification for user to supervisor messages. You can use a mobile web browser if you need to act as a supervisor on a mobile device.)

You may want to unsubscribe from notifications for the user channels, because the supervisors take turns communicating with all users. See chapter “Unsubscribe from terminal channels” if for details.

If you become supervisor (by receiving permission useAllChannels) or re-subscribe to a channel, you get access to all historic messages in the channel. All messages written within during the last 24 hours are considered unread.

Terminal channel

There is one terminal channel per depot which acts as a broadcast channel. You can access it if you have been granted access.
If you can post a message, all participants – all users with the permission useTerminalChannel – will see it.
Your message may be text. In cadisAPP, you may additionally send a photo. In cadisSTUDIO, you may additionally send image files.
You will be notified about all messages from other participants and can read all their messages.

Unsubscribe from Terminal channel notifications

You may unsubscribe from notifications about new messages in the terminal channel. You will still retain access to the terminal channel and see markers for new messages there.

click the bell
click settings

uncheck “Messages in the terminal channel”
click Save.

Vehicle channels

Note: The "Vehicle channel" functionality is unchanged from the long established "Vehicle messages" functionality on cadisAPP, cadisSTUDIO and cadisCONTROL. If the new extended "Message channels" functionality is not enabled you will see no difference to the user interface.

In cadisAPP, if you have logged in for a certain vehicle, click on the icon “Messages” to get access to your vehicle messages.

As a supervisor having logged in in cadisSTUDIO, you have access to all vehicle channels. You may unsubscribe from vehicle channel notifications. See chapter “Unsubscribe from terminal channels”.
