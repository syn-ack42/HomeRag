CADIS Knowledge Base : Display and edit vehicle combination for a loading

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

The vehicle combination of a loading can be set or changed. This is visible in “My loadings” for a loading list in the respective details and can also be edited depending on the status of the loading list.

View vehicle combination

The vehicle combination for a loading list can be shown in the “Info” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Info”

  4. View “Vehicle combination”

Edit vehicle combination

The vehicle combination for a loading list can be edited in the “Info” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Info”

  4. View “Vehicle combination” button

  5. Click “Edit” button

  6. Modify vehicle combination

  7. Click “Save” button

To select a vehicle, you have different options:

Option 1: Auto Completion

You can search for the:

  • Id of the vehicle

  • License plate of the vehicle

  • Type of the vehicle (e.g. Truck)

Option 2: Manual selection

  1. Click “Open” button

  2. Select “Vehicle types” or “Vehicles”

  3. Select a vehicle

  4. Click “Select” button
