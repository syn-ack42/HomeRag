General

The vehicles table can be accessed via the "First & Last Mile" tab "Vehicles" on the cadisSTUDIO tile.

The List of vehicles shows all vehicles that match the selected filter criteria. It shows important information at a glance, such as which vehicles already have a cadisAPP registered (including the driver if applicable), which vehicles are currently loaded and on tour, whether a data connection exists, and whether GPS positions are being transmitted. Additionally, the overview contains a summary of all running collection and delivery orders for the current day per vehicle.

You can filter, select, and group vehicles as desired.

Filters

Data Query Filters

In the left, basic filter area, various filter options are available in a combo box to specifically adjust the vehicle overview and find relevant information faster.

The following filter options are available:

  • Available – Vehicles are displayed here as available or unavailable based on the stored vehicle master data. If a vehicle is only partially available, this is marked in the "Available" column with a clock (insert icon). In the filter selection, such vehicles are rated as "available".

  • Orders – Enables filtering of vehicles that already have orders assigned or are currently without orders.

  • Radio status – Indicates whether the driver's mobile device currently has a connection, has a connection problem, or no connection exists.

  • Trip status – Shows whether a vehicle is currently in the depot, on a tour, or logged out.

  • Vehicle group – Enables filtering by a specific dispatch group. These groups are assigned to vehicles in the master data and allow structured management according to specific areas of use or responsibilities.

Additionally, multiple filters can be used simultaneously to enable targeted narrowing of the displayed vehicles.

Above the table, the currently active filters are displayed, so it is immediately apparent which filter settings are active. The basic filters selected on the left side are highlighted in yellow.

Toolbar "Columns" and "Quick Column Filter"

To the right of the table in the vertical tab is the Columns block. In this area, all columns are displayed.

To the right of the table, you can also use the vertical tab Filter to further restrict the vehicle quantity by entries of each column as desired. Above the table, you can see the currently active filters: in white the further restrictions chosen on the right.

Table

An explanation of the individual columns is displayed as a tooltip when you hover over the respective column header with the mouse and pause there.

Some columns in the table are thematically grouped. These groups include, among others, delivery orders, pickup orders, and messages. Each of these categories contains specific information that enables a structured and clear presentation of the data.

In the columns "Orders to this vehicle", "Delivery orders to this vehicle", and "Collection orders to this vehicle", there are interactive links through which you can directly access the Orders table. The corresponding filters are automatically applied so that only the relevant orders for the respective vehicle are displayed.

The table also contains a summary row that sums all numerical values. Boolean values are also counted.

Context Menu for Vehicles

  • Vehicle Details – This option opens the master data of the selected vehicle. This includes information such as vehicle type, license plate, mileage, maintenance status, and other relevant vehicle data.

  • Show Orders for this Vehicle – Displays all orders linked to the chosen vehicle in the order overview. (https://cadissoftware.atlassian.net/wiki/x/AQAuUw )

  • Send Messages – Opens the message window to send a message to the selected vehicle. Through multiple selection, the message can be sent to several vehicles at once. (https://cadissoftware.atlassian.net/wiki/x/UwAwT )

  • Show Vehicle on Map – This function opens the map window and shows the exact position of the selected vehicle. It allows you to visually locate the vehicle on the map and, if necessary, check further details such as current movement or location history.

  • Manage mobile access – When this option is selected, the "Users and Devices" area opens under the menu item "Mobile access". Here you can either create a new authentication for a vehicle or modify an existing access.

Additional Functions

Vehicle Quick-View

When exactly one vehicle is selected in the vehicle overview, the quick view display opens with detail information such as vehicle, driver, payload, tour status, radio time, radio status, GPS time. Additionally, for pickup and delivery orders, the following status information is displayed: Running, time-critical, completed, with obstacle, total count.

Direct Links to Order Overview

In several columns (Number of Orders, Number of Collections, Number of Deliveries), the displayed value is linked. Through this link, you can directly open the order overview filtered to exactly these orders.
