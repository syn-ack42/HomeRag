CADIS Knowledge Base : The Notes

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Notes can be created & processed between departments members like dispatchers or customer service.

How to create a note

❶ Click on New

❷ Enter your note

❸ Select the recipient

❹ Click OK

❺ Note intended for customer service created with author (user) name & creation date

How to view the unprocessed Notes

❶ After a note is created, cadisCONTROL shows a notification to the attention of users

❷ If you click on the notification, you will be driven to a view of the list a shipments showing only the shipments for which a note must be processed

❸ Columns exist to indicate that notes are waiting to be processed & since when

❹ Double click on a shipment to process the note

How to process a Note

❶ Double click on a note to process it

❷ After a user has opened the note, a time stamp is showed in the field “Read”

❸ Click on process

❹After a user has processed the note, a time stamp is showed in the field “Done”
