CADIS Knowledge Base : Transfer Column & Filter Settings in cadisControl to Another Depot

Created by Julian Beyer, last modified on Sep 08, 2025

Saving Work Environments as Views

In almost every dialog, you can apply various filters or customize the column layout. For example:

  • Show/hide columns

  • Change the order of columns

  • Adjust sorting per column

A work environment configured in this way – consisting of customized column and filter settings – can be saved for quick access later. If multiple views are saved, you can easily switch between them. When saving, you can choose whether all users in the same depot should also be able to use this view (see screenshot below).

Transferring Views

Saved views that are available to all users of the same depot can be exported and imported into another depot. This is possible both within the same environment and between different environments (e.g., from TEST to PROD). You can access the export/import function via the menu "Extras → User Settings" (see screenshot below).

If there are major differences in the software version between environments, there may be limitations when transferring views.
