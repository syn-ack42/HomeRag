CADIS Knowledge Base : Message Overview

Open the Message overview using the :nachrtichtenUebersicht: symbol in the cadis control panel.

The message overview shows all text messages that were exchanged between dispatchers and vehicles (drivers).

The Filter area

Selects which messages should be displayed.

Table

The messages are displayed here in short form with the following columns

  • Origin
    Symbol for the sender of the message

  • Author
    User name, if the message was sent by the dispatching department
    Vehicle number, if the message was sent by the driver.

  • Vehicle
    Vehicle number with whom the message was exchanged

  • Time
    Time when the message was sent

  • New
    Indication that the message has not yet been read

  • Trans
    Indication that the message has not yet reached the mobile system (only for outgoing messages)

  • Message
    Beginning of the message text- The complete message is visible in the detail window.

The Context Menu

Using the context menu (pressing the right mouse button on the table) you can carry out various actions.

  • Message details
    Displays the message details, so the message is considered as read.The message details are also available using the :pfeilRechtsOben: button in the message line.

  • Use the arrow keys to go to the previous / next message.

  • Delete
    Deletes the selected messages

  • Answer
    Opens the window for sending a message

  • Marked as read
    Deletes the new marker
