CADIS Knowledge Base : Freight carrier selection for loadings

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

A cadis user has the possibility to assign a freight carrier to a last mile or line haul loading.

Therefore, different options exist.

Option 1:

A vehicle is assigned to the loading. For that, the user has to edit the vehicle combination. If a freight carrier is assigned to the vehicle in its master data, this connected freight carrier data is automatically taken over into the loading. If the assigned vehicle gets changed, the freight carrier is overwritten with the new freight carrier data.

Option 2:

The user can directly select a freight carrier from the master data.

Option 3:

The user can edit the freight carrier data separately and enter these data manually in an own dialog. Editing is only possible, when existing data (for example from the vehicle or a master data selection) is deleted before.

Notice: This behavior to select a freight carrier has been standardized and is also used for "Trip templates" and "My trips".
