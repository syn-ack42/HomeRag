CADIS Knowledge Base : My Unloadings

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

General

In the section „My Unloadings” unloading lists are displayed in an overview on a daily basis. The unloading list details can be viewed and (partially) edited here as well as functions can be carried out on unloading list level.

Licenses, configuration and permissions

  • License for terminal handling

Overall view

Display of the unloading lists

List element

For every unloading list a history is available.

Filter and grouping of the unloading lists

Standard filter: Current date, selectable on a daily basis

Traffic type: All unloading lists can be displayed or optionally only those for last mile or line haul.

Sort unloadings (ascending and descending):

  • Status

  • Arrival time

  • gate

Free filter input:

  • License plate

  • Name of the loading list

  • Outbound route

  • Vehicle ID

Display of the filter

  • In white ellipses: Filter is allowed to be set in this view

  • Is a filter displayed in a yellow ellipse, than this filter cannot be set manually. It was set automatically by selecting this view from another view.

An ascending respectively descending sorting of the filtered values is also possible.

Display of the shipments in an unloading list

Header data of the unloading list

Display of essential unloading list data

When you select the header data of the unloading list, the associated shipments and the unloading list details are displayed in the right area.

In case of multi selection, there is no detailed display – instead, the user receives information about how many unloading lists were selected.

When you select a shipment from the unloading list, the shipment details are displayed in the righthand area and the option to set a status is offered in the action menu.

Multi select is possible – in this case setting the status is done for all selected shipments.

Setting the status is also possible on package level and for multiple selection.

Shipment filter

Here you can filter by shipment number using free text input.

Criteria for filtering

Status filter: OR junction

Attributes: AND junction

Unloading list details

In addition to general information about the unloading list, a user can also view and edit the vehicle combination.

Currently valid gates and seals can be maintained here.

Unloading equipment can be added.

The unloading report can be maintained.

Shipment list

The essential data is displayed for each shipment.

When you select the shipment, the shipment details are displayed in the right area.

Functions

You can write messages for this vehicle.

You can release the unloading list for unloading.

As soon as unloading is in progress, the function for closing the unloading list is offered.

Error handling during closing:

  • Unloading checks are displayed

  • Errors are reported and must be corrected in the planning module (My Trips).
