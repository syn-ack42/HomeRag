Open the List of vehicles using the :FahrzeugUebersicht: symbol in the cadis-control panel.

The List of vehicles shows all vehicles that match the selected filter criteria. The overview is updated automatically every 30 seconds, but can also be updated manually using the :refresh: symbol.

Filter area

  • Route
    This filter currently has no effect

  • Route status
    Filters only vehicles with a specific route status (On depot, On tour, Logged off)

  • Vehicle group
    Filters according to the dispatching group (Assignment to a dispatching group takes place in the resource master data/vehicles)

  • Radio status
    Filters according to the radio communication status of the mobile system (OK, Error, No connection)

  • Only available
    Displays only vehicles that are marked as available in the master data

  • Only with orders
    Displays only vehicles to which orders are currently assigned.

Table

Shows an explanation for the columns as a tooltip, if you hover with the mouse on the column header

  • Columns highlighted in red contain collection data

  • Columns highlighted in blue contain delivery data

  • Columns highlighted in green contain messages data

  • Columns Release ok / Release to: see below: Vehicle Authentication

Context menu

Vehicle Details

Displays the vehicle master data for the selected vehicle

Collection / Delivery orders for the selected vehicle

Displays a lists of orders, collections and deliveries for the selected vehicle

Send Messages

Opens the window to send messages to the selected vehicle(s)

Display Vehicle On Map

Opens the map window and shows the position of the selected vehicle

Display Route

Opens the map window and shows the position and the route of the selected vehicle

Display Mobile Device

Opens the list of mobile devices in cadisSTUDIO (Mobile device management) and displays the mobile device data of the selected vehicle.

Edit Mobile Access

If selected, the section "Users and devices" / "Mobile accesses" in cadisSTUDIO is opened. The authentication for a vehicle can be created or changed. The role has to be entered obligatory (Driver, MobileUser, TerminalWorker).

Click on "Generate" to generate a barcode with which a mobile device can register as a vehicle.

Delete Mobile Access

It is possible to delete the mobile access via overview. Authentication is deleted, i.e. the registration of the mobile device and the registration barcode become invalid.

Print registration barcode

A PDF document with the registration barcode is generated and displayed which you can then print or save and for example send by email.
