CADIS Knowledge Base : Display and edit loading equipment for loading

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Loading equipment can be recorded for a loading. These are visible in “My loadings” for a loading list in the respective details and can also be edited depending on the status of the loading list.

View loading equipment

The loading equipment for a loading list can be shown in the “Loading equipment” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Loading equipment”

Add loading equipment

The loading equipment for a loading list can be added in the “Loading equipment” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Loading equipment”

  4. Click “Edit” button

  5. Modify loading equipment

  6. Click “Save” button

Edit loading equipment

The loading equipment for a loading list can be edited in the “Loading equipment” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Loading equipment”

  4. Click “Edit” button

  5. Modify loading equipment

  6. Click “Save” button
