CADIS Knowledge Base : Last Mile Routes

Created by Julian Beyer, last modified on Aug 21, 2025

You can reach the screen for creating Last mile routes by using the master data overview (:regeldefinition: symbol in the cadis control panel).

Last mile trips are always created for a specific last mile route from the master data. Such a route can, for example, define a geographical area or be used to group certain shipments.

The fields marked with an asterisk are mandatory.
