Vous accédez à la fenêtre «  Liste des véhicules » en cliquant sur le symbole :FahrzeugUebersicht: dans la barre des tâches cadis.

La fenêtre « Liste des véhicules » affiche l’ensemble des véhicules qui répondent aux critères de filtrage sélectionnés dans la partie « Filtres ». La liste est automatiquement mise à jour toutes les 30 secondes mais elle peut être mise à jour manuellement en cliquant sur le symbole :refresh:.

Filtres

  • Tournée
    Inutilisé à ce jour

  • Statut Tournée
    Filtre uniquement les véhicules ayant un statut de tournée (Agence, Tournée, Déconnecté)
    Ce filtre est un choix sur liste.

  • Groupe de véhicules
    Filtre en fonction du groupe de véhicules (l’affectation à un groupe de véhicules est effectuée en prenant en compte les informations présentes dans les référentiels ressources/véhicules)
    Ce filtre est un choix sur liste.

  • Statut Communication
    Filtre en fonction des statuts de la communication radio (OK, Perturbation, Non connecté)
    Ce filtre est un choix sur liste.

  • Uniquement véhicules disponibles
    Affiche uniquement les véhicules qui sont indiqués comme disponibles dans les référentiels
    Ce filtre est une case à cocher.

  • Uniquement avec ordres
    Affiche uniquement les véhicules auxquels des ordres sont actuellement attribués.
    Ce filtre est une case à cocher.

Authentification du véhicule

Dans la fenêtre « Liste des véhicules », vous gérez l’authentification des appareils mobiles pour les conducteurs, c’est-à-dire pour les appareils mobiles qui se connectent à cadis avec un numéro de véhicule (plaque minéralogique).

Pour activer cette fonction, cochez la case « Afficher l’authentification ». Après avoir accepté que l’activation peut ralentir les performances de cadisCONTROL, les colonnes « Code ident OK » et « Connexion autorisée jusqu’au » du tableau affichent les données associées.

Vous disposez aussi de l’accès aux menus contextuels dans le tableau (clic droit), ils permettent de visualiser et de modifier l’authentification des utilisateurs mobiles en ouvrant la fenêtre « Authentification ».

Tableau

Les textes et symboles affichés dans les entêtes des colonnes précisent le contenu de ces colonnes ; des info-bulles plus détaillées apparaissent lorsque vous survolez ces entêtes avec votre souris.

  • Les entêtes des colonnes sur fond rouge indiquent que les données de ces colonnes concernent les enlèvements

  • Les entêtes des colonnes sur fond bleu indiquent que les données de ces colonnes concernent les livraisons

  • Les entêtes des colonnes sur fond vert indiquent que les données de ces colonnes concernent les messages

  • Les colonnes Connexion ok / nok, Connexion autorisée jusqu’au : voir section Authentification du véhicule ci-dessous:

Menu contextuel

Le menu contextuel (clic droit dans le tableau) vous permet d’effectuer diverses actions sur les expéditions sélectionnées.
Les actions qui sont affichées en gris ne peuvent pas être utilisées pour les expéditions sélectionnées ; la raison de l’interdiction d’utilisation apparait sous forme d’une info-bulle lorsque vous passez la souris sur le texte grisé.

Détails véhicule

Ouvre la fenêtre « Véhicules » et affiche les données (issues des référentiels) du véhicule sélectionné.

Ordres d’enlèvement / de livraison correspondant à ce véhicule

Ouvre la fenêtre « Suivi des ordres » et affiche la liste des enlèvements et/ou des livraisons du véhicule sélectionné.

Envoyer un message

Ouvre la fenêtre « Envoyer un message » et vous permet d’envoyer un message à un véhicule (par défaut le véhicule sélectionné), plusieurs véhicules ou un groupe prédéfini de véhicules.

Afficher le véhicule sur la carte

Ouvre la fenêtre « Géolocalisation » et affiche la position du véhicule sélectionné sur la carte.

Afficher la route

Ouvre la fenêtre « Géolocalisation » et affiche la position du véhicule sélectionné ainsi que l’itinéraire déjà réalisé.

Afficher le terminal mobile

Ouvre la fenêtre « Suivi des terminaux mobiles » et affiche les données concernant le terminal mobile associé au véhicule sélectionné.

Détails d’authentification

Ouvre la fenêtre « Authentification » et vous permet de créer ou modifier les informations liées à l’authentification d’un véhicule ainsi que de générer un code-barres avec lequel un appareil mobile peut s’enregistrer avec cadisAPP en tant que véhicule dans cadis.

Nécessite d’avoir coché la case « Afficher l’authentification » dans les filtres (en haut de la fenêtre).

Retirer code-barres d’identification

L’authentification est retirée, la connexion de l’appareil mobile et le code-barres de la connexion ne sont plus valides.

Imprimer le code-barres d’identification

Un document PDF contenant le code-barres de l’enregistrement est généré et affiché. Vous pouvez directement l’imprimer, ou l’enregistrer voire l’envoyer par email.
