CADIS Knowledge Base : How long do orders and stops stay visible on cadisAPP?

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Orders (and their stops) are removed from cadisAPP:

  • When the user processes the order (sets a status) and ends the tour or logs off the device.
    (As a rule, the system is configured so that “End tour” or “Log off user” is only possible when all orders have been processed).

  • If the order has a final status and data is synchronized with the server. Only orders that can still be processed are synchronized.
    This is done by:

    • manual data synchronization

    • When logging on to the same release with another device (transfer of the session and thus the tour)

    • After a serious crash.

  • After a serious crash.

  • When a final status is set by the control station.

  • When the order is canceled by the scheduling department (order cancellation).

  • When a shipment is removed from the tour by importing data from the host system.

  • If the shipment is completely canceled

  • If a delivery is transferred to another tour by “Handover on Tour”.

  • When the entire tour is completed or canceled.
