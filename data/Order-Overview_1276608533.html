CADIS Knowledge Base : Order Overview

Created by Julian Beyer, last modified on Aug 19, 2025

Open the Order overview using the :AuftragUebersicht: symbol in the cadis control panel.

The order overview displays all collection and delivery orders that have been assigned for processing in a secondary distribution route.

Scheduled shipments and collections are converted into delivery and collection orders for processing on the secondary distribution route. In this way, for example, a shipment that could not be delivered at the first delivery attempt (first delivery order) can be delivered the next time: A second delivery attempt is carried out with a second delivery order.

Important: Collection and delivery orders for a route are only visible when loading is completed and the route is released.

If a collection is cancelled or rescheduled, the original collection orders are maintained but are given the status “Cancelled”.

The overview is updated automatically every 30 seconds, but can also be updated manually using the :refresh: symbol.

Register

By selecting a register (tab sheet), you can choose which orders are to be displayed:

Collection orders, collection and delivery orders, delivery orders.

Filter Area

To display as many shipments as possible, you can hide the filter area and show it again using the :FilterDown: symbol.

You can filter the orders according to different criteria:

  • Vehicle Group: A dispatching group that is defined using the entry in the vehicle master data

  • Vehicle: A vehicle can be selected from the master data

  • Route: Selection from the route master data, filters the orders according to the outbound relationship

  • Status: Filters according to order status

  • Attribute: Filters according to order characteristics

Table

The columns are self-explanatory. You can display the description of the column as a tooltip by hovering the mouse over the column header. In many cases, if the field content is unclear, you can use the tooltip to obtain further details.

The status times that are marked in yellow to indicate that the status was assigned on a different day than the one set in the filter area.

Context menu

Using the context menu (press the right mouse button on the table) you can perform different actions on the selected shipments.

  • Details
    Displays the order details.

  • Change status
    You can change or set the status of the order if the mobile device has transferred an incorrect status or no status at all

  • Set processing status
    Sets a processing status to document processing steps that take place outside cadis.
    The processing status is displayed in the overview in the Processing Status column.

  • Confirm time critical
    By doing this you acknowledge when an order has become time-critical. It then no longer appears in the Time critical order filter (not confirmed)

  • Show order on map
    Opens the map window on which the selected order is displayed.
