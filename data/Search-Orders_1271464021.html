CADIS Knowledge Base : Search Orders

Created by Julian Beyer, last modified on Sep 08, 2025

Open the Search orders screen using the :AuftragSuche: symbol in the cadis control panel.

In this screen you can search for orders using different criteria.

For this, select the corresponding tab for collections or deliveries, enter your search criteria in the corresponding fields and click on the search button.

The table and the actions that you perform using the context menu match the order overview.
