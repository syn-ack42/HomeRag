CADIS Knowledge Base : Terminal Locations

Created by Julian Beyer, last modified on Aug 21, 2025

Open the screen for creating Terminal areas by using the master data overview (:regeldefinition: symbol in the cadis control panel).

A terminal is subdivided into areas. Each area contains terminal locations. They serve to improve the organisation within the terminal. For example, it is easier to find a package if the location in which it was placed has been recorded or in which area of the hall it can be found.

The fields marked with an asterisk are mandatory.
