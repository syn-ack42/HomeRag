CADIS Knowledge Base : Transporteur

Created by Julian Beyer on Aug 21, 2025

A partir de la page « Référentiels » de cadisSTUDIO vous sélectionnez dans la colonne de gauche « Transporteur ».

Vous pouvez alors, créer, modifier ou supprimer un transporteur en le sélectionnant (création/modification/suppression accessible selon vos droits utilisateurs et le type de données centrales/locales).

Seuls les champs marqués d’un astérisque sont obligatoires.

À noter : le numéro d’identification ID doit être unique.

En cliquant sur Enregistrer, vous sauvegardez les modifications ; si vous cliquez sur Annuler, elles sont supprimées.

Il n’est pas nécessaire de créer les référentiels « Transporteur » pour utiliser toutes les fonctions cadis.
