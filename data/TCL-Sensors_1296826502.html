CADIS Knowledge Base : TCL Sensors

Created by Julian Beyer, last modified on Aug 19, 2025

Take a look at your sensor data by clicking on “Sensors” in TCL.

This sensor overview lists all sensors for which cadis has received data from the configured telematic provider.

Sensor data received from the provider:

  • A Sensor is identified by the triple {Telematic Provider, Data Logger, Sensor}.

  • The Temperature is the latest value received from the telematic provider at the Timestamp shown next to it.

Data originating from cadis:

  • The Compartment (not implemented yet) is the terminal compartment for which the sensor has been chosen as relevant sensor, and the vehicle compartment that has been scanned in cadisAPP.

If you click on a sensor, cadis shows the temperatur values of this sensor in the last 24 hours in a graphics in a window to the right.

A double click on a sensor opens a detailed graphics where you can choose the time interval of interest.
