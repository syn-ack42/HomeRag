CADIS Knowledge Base : Fahrzeugkombinationen in der Beladung anzeigen und bearbeiten

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Die Fahrzeugkombination einer Beladung kann angezeigt und geändert werden. Die Fahrzeuge sind in “myLoadings” in den Details der Ladeliste sichtbar und können je nach Status der Ladeliste auch bearbeitet werden.

Fahrzeugkombination anzeigen

Die Fahrzeugkombination für eine Ladeliste wird im Widget “Info” angezeigt.

  1. Öffnen Sie “myLoadings”.

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Info”

  4. Ansicht “Fahrzeugkombination”

Fahrzeugkombination bearbeiten

Die Fahrzeugkombination für eine Ladeliste kann im Widget “Info” bearbeitet werden.

  1. Öffnen Sie “myLoadings”

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Info”

  4. Schaltfläche “Fahrzeugkombination” anzeigen

  5. Schaltfläche “Bearbeiten” anklicken

  6. Fahrzeugkombination bearbeiten

  7. Klicken Sie auf die Schaltfläche “Speichern”.

Bei der Auswahl eines Fahrzeugs haben Sie verschiedene Möglichkeiten:

Option 1: Automatische Vervollständigung

Sie können suchen nach:

  • ID des Fahrzeugs

  • Nummernschild des Fahrzeugs

  • Typ des Fahrzeugs (z. B. Lkw)

Option 2: Manuelle Auswahl

  1. Schaltfläche “Öffnen” anklicken

  2. Wählen Sie “Fahrzeugtypen” oder “Fahrzeuge”.

  3. Wählen Sie ein Fahrzeug

  4. Schaltfläche “Auswählen” anklicken
