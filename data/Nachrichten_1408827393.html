CADIS Knowledge Base : Nachrichten

Created by Sergov, Lena, last modified by Julian Beyer on Aug 21, 2025

Sie erreichen den Bereich Nachrichten in cadisSTUDIO über die Kachel

Links in der Übersicht sind alle vorhandenen Chats zu finden. Nach Selektion kann ein bereits begonnener Chat direkt in der Übersicht fortgesetzt werden.

Die Dialoge können nach Datum gefiltert werden.

Über den Button Neu starten Sie eine neue Kommunikation. Sie können wählen zwischen Halle, Fahrzeug, Fahrzeuggruppe und Benutzer.

Nach Auswahl öffnet sich ein Dialog, in dem zum einen die Nachricht selbst eingegeben wird und zum anderen die Fahrzeuge bzw. die Fahrzeuggruppe (Nachricht wird an alle Fahrzeuge der ausgewählten Fahrzeuggruppe gesendet) per Checkbox ausgewählt werden.

Über den Button Senden wird die Eingabe bestätigt und der Chat gestartet.

Alternativ kann auch an eine Tour ein Chat gestartet werden (Voraussetzung: Tour ist zur Abfahrt freigegeben): Bereich Umschlagshalle -> Meine Beladungen -> Tour auswählen -> Nachricht schreiben
