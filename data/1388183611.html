CADIS Knowledge Base : How can I recognize if there´s existing a referenced shipment to a shipment / collection / partial shipment?

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

  • Shipment details -> Tab “status” -> referenced shipment is displayed

  • Shipment overview: Search for global tracking number -> Both shipments are displayed
