CADIS Knowledge Base : Référentiels

Created by Julian Beyer on Aug 21, 2025

Vous accédez à la page web présentant les masterdata / « Référentiels » :

  • Soit en cliquant sur le symbole :regeldefinition: dans la barre des tâches cadisCONTROL, puis en sélectionnant « Référentiels (cadisSTUDIO) » dans la liste de choix.

  • Soit en ouvrant directement cadisSTUDIO dans le navigateur web.

Dans les deux cas, une page d’identification apparait (utilisant le navigateur web par défaut), elle vous permet de vous identifier à cadisSTUDIO.

Une fois cette identification réalisée, vous accédez à la page web « Référentiels » de cadisSTUDIO

Cette page vous permet de créer et de mettre à jour les référentiels concernant les référentiels.

La colonne de gauche vous permet de créer / modifier / supprimer des données associées aux référentiels en sélectionnant le type de MasterData concerné.

Quel que soit le référentiel sélectionné dans la colonne de gauche :

Le bouton créer permet de créer une nouvelle ligne, donc une nouvelle entrée dans le référentiel sélectionné.

Les boutons Supprimer et Ouvrir effectuent l’action sur la ligne sélectionnée. La modification est accessible par double clic sur une ligne ou clic sur Ouvrir.

Utilisation

Les référentiels que vous créez ne sont valides que dans l’agence où vous les créez.

À noter : Certains référentiels sont valides pour l’ensemble des agences ou pour un sous-ensemble de ces agences si les données ont été importées. L’information du champ d’application (agence/agences) apparait dans la colonne Agence. Il n’est possible de modifier ou de supprimer que les données locales à l’agence (créées par cadisSTUDIO), les données importées ne sont ni modifiables ni supprimables.
