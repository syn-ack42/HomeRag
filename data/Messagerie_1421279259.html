CADIS Knowledge Base : Messagerie

Created by Julian Beyer on Aug 20, 2025

Vous accédez à la fenêtre « Messagerie » en cliquant sur le symbole :nachrtichtenUebersicht: dans la barre des tâches cadis.

La fenêtre « Messagerie » liste les messages échangés entre le service d’exploitation cadisCONTROL et les conducteurs (véhicules) cadisAPP.

Barre d’icônes

:refresh:: En cliquant sur ce bouton, une mise à jour des données s’effectue et les nouvelles données s’affichent dans la fenêtre.

:kopieren:: En cliquant sur ce bouton, cadis copie les données sélectionnées dans le tableau dans le presse-papiers pour réutilisation.

Filtres

Vous filtrez les échanges en fonction de plusieurs paramètres (numéro de véhicule, sens de transmission, etc…) qu’il est possible de combiner entre eux ; pour réinitialiser ces filtres repositionnez les paramètres à la valeur par défaut « Tous ».

Tableau

Un résumé de chaque message émis ou reçu est affiché sur chaque ligne, les colonnes détaillent les informations liées aux messages :

  • Origine
    Affiche un symbole représentant l’expéditeur du message – véhicule ou exploitant -.

  • Auteur
    Affiche le nom d’utilisateur, si le message a été envoyé par le service d’exploitation (cadisCONTROL) ou le numéro du véhicule, si le message a été envoyé par l’application mobile (cadisAPP).

  • Véhicule
    Affiche le numéro du véhicule avec lequel le message a été échangé.

  • Heure
    Affiche l’heure à laquelle le message a été envoyé.

  • Nouveau
    L’icône lorsqu’elle est présente, indique que le message n’a pas encore été lu par le destinataire.

  • Transmission
    L’icône lorsqu’elle est présente, indique que le message n’a pas encore été reçu par le terminal mobile / cadisAPP (s’affiche uniquement pour les messages sortants).

  • Message
    Affiche le début du texte du message – Le message complet est visible dans la fenêtre «  Détails » expliquée ci-dessous -.

Menu contextuel

Le menu contextuel (clic droit dans le tableau) vous permet d’effectuer 4 actions pour le/les messages sélectionnés.

  • Détails
    Affiche les détails du message, le message est ensuite considéré comme lu.
    Les détails du message sont également disponibles via le bouton :pfeilRechtsOben: directement à partir de la ligne du message.

  • Les touches flèche droite / flèche gauche vous permettent de naviguer entre les messages sélectionnés.

  • Supprimer
    Supprime totalement le/les messages sélectionnés.

  • Répondre
    Ouvre la fenêtre « Envoyer un message » et vous permet d’envoyer un message de réponse.

  • Marquer comme lu
    Fait disparaitre l’icône non lue et signale que le message a été lu.
