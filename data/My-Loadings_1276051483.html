General

In the section „My Loadings” loading lists are displayed in an overview on a daily basis. The loading list details can be viewed and (partially) edited here as well as functions can be carried out on loading list level.

Licenses, configurations and permissions

  • License for terminal handling

  • License for area freight forwarder (if it relevant to be filtered)

  • “closeDepartureLocalTraffic” for „Close loading”-Button

  • “viewTransportsLocalTraffic” for switch to MyTours

  • “readMessage” and “changeMessages” for „Messages“-Button

Overall view

Display of the loading lists

List element

For every loading list a history is available.

Filter and grouping of the loading lists

Standard filter: Current date, selectable on a daily basis

Traffic type: All loading lists can be displayed or optionally only those for last mile or line haul.

Sort loadings (ascending and descending):

  • Status

  • Departure time

  • gate

Free filter input:

  • License plate

  • Name of the loading list

  • Outbound route

  • Vehicle ID

Display of the filter

  • In white ellipses: Filter is allowed to be set in this view

  • Is a filter displayed in a yellow ellipse, than this filter cannot be set manually. It was set automatically by selecting this view from another view (for example selection in the dashboard)

An ascending (respectively descending) sorting of the filtered values is also possible.

Display of the shipments in a loading list

Header data of the loading list

Display of essential loading list data

When you select the header data of the loading list, the associated shipments and the loading list details are displayed in the right area.

In case of multi selection, there is no detailed display – instead, the user receives information about how many loading lists were selected.

When you select a shipment from the loading list, the shipment details are displayed in the right area and the option to set a status and to print delivery note is offered in the action menu.

Multi select is possible – in this case setting the status respectively the print of the delivery note is done for all selected shipments.

Setting the status is also possible on package level and for multiple selection.

Shipment filter

Here you can filter by shipment number using free text input.

Critera for filtering

Status filter: OR junction

Attributes: AND junction

Loading list details

In addition to general information about the loading list, a user can also view and edit the vehicle-combination.

The planned terminal location and handling instructions can be set via the dispatching info.

Currently valid gates and seals can be maintained here.

Loading equipment can be added.

The loading report can be maintained.

Shipment list

The essential data is displayed for each shipment.

When you select the shipment, the shipment details are displayed in the right area.

Functions

If you have the permissions to edit atrip, you can switch to “My Trips” here.

You can write messages for the assigned vehicle.

You can release the loading list for loading.

As soon as loading (driver loading) is in progress, the function for closing the loading list is offered.

Error handling during closing:

  • Loading checks are displayed

  • Errors are reported and must be corrected in the planning module (My Trips).
