CADIS Knowledge Base : Auswahl des Frachtführers für Beladungen

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Ein Benutzer hat in cadis die Möglichkeit, einer Nah- oder Fernverkehrsbeladung einen Frachtführer zuzuweisen.

Hierfür gibt es verschiedene Optionen.

Option 1:

Der Beladung wird ein Fahrzeug zugewiesen. Dazu muss der Benutzer die Fahrzeugkombination bearbeiten. Ist dem Fahrzeug in seinen Stammdaten ein Frachtführer zugewiesen, werden diese verknüpften Frachtführerdaten automatisch in die Beladung übernommen. Wird das zugewiesene Fahrzeug geändert, dann wird der Frachtführer mit den neuen Frachtführerdaten überschrieben.

Option 2:

Der Benutzer kann direkt einen Frachtführer aus den Stammdaten auswählen.

Option 3:

Der Benutzer kann die Frachtführerdaten separat bearbeiten und diese manuell in einem eigenen Dialog eingeben. Das Editieren ist erst möglich, wenn existierende Daten (z.B. aus dem Fahrzeug oder einer Stammdatenauswahl) vorab gelöscht wurden.

Hinweis: Das Verhalten zur Auswahl eines Frachtführers wurde standardisiert und wird auch für die "Regeltouren" und "Meine Touren" verwendet.
