CADIS Knowledge Base : Tore in der Beladung anzeigen und bearbeiten

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Zu einer Verladung können Tore erfasst werden. Diese sind in “myLoadings” zu einer Ladeliste in den Details sichtbar und können je nach Status der Ladeliste auch bearbeitet werden.

Tore ansehen

Die Tore einer Ladeliste können im Widget “Tore” angezeigt werden.

  1. Öffnen Sie “myLoadings”

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Tore”

Tore hinzufügen

Die Tore für eine Ladeliste können im Widget “Tore” hinzugefügt werden.

  1. Öffnen Sie “myLoadings”

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Tore”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Tor ändern

  6. Klicken Sie auf die Schaltfläche “Speichern”.

Gates bearbeiten

Die Gates einer Ladeliste können im Widget “Tore” bearbeitet werden.

  1. Öffnen Sie “myLoadings”

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Tore”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Tor ändern

  6. Klicken Sie auf die Schaltfläche “Speichern”.
