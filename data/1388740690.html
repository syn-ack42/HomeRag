CADIS Knowledge Base : Wie funktionieren die Tabellenfilter in cadisSTUDIO?

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Tabellen in cadisSTUDIO haben zwei Arten von Filtern: "Datenabfragefilter" und "Schnellfilter".

Datenabfragefilter

Datenabfragefilter werden auf die Daten angewendet, wenn sie vom cadisSERVER aus der Datenbank abgerufen werden. Wenn ein Datenabfragefilter geändert wird, werden die Daten immer vom Server neu geladen. Bei vielen Tabellen in cadis ist das Setzen eines Datenabfragefilters verpflichtend, um die Anzahl der übertragenen Zeilen auf der cadisSTUDIO-Seite zu begrenzen.

Welche Spalten gefiltert werden können und ob überhaupt ein Datenabfragefilter vorhanden ist, hängt von der jeweiligen Tabelle und den zugehörigen Anwendungsfällen ab.

Der Tabellenfuß in cadisSTUDIO zeigt immer eine Zeilenanzahl in der Form "Zeilen: XX von YY" an. Die Zahl "YY" gibt an, wie viele Zeilen mit dem angewendeten Datenabfragefilter in der Datenbank gefunden wurden.

Datenabfragefilter werden über den Block auf der linken Seite neben der Tabelle eingestellt. Sie können Filteroptionen über das "+"-Symbol hinzufügen. Die Einstellungsmöglichkeiten für diesen Filter erscheinen oben im Block. Ein Filter kann mehrere Optionen haben (z.B. Name und Postleitzahl für den "Empfänger"). Sobald ein Datenabfragefilter angewendet wird, erscheint er als größere orangefarbene Blase über der Tabelle.

Für jeden Datenabfragefilter können Sie auf das "Durchgestrichener Trichter"-Symbol klicken, um den Filter zu leeren, oder auf das "Mülleimer"-Symbol, um ihn vollständig zu entfernen. Einige Filtertypen bieten auch ein "Ausgefüllter Trichter"-Symbol. In diesem Fall hat der Filter einen vordefinierten Standardwert (z.B. "heute"). Setzen Sie diesen Standardwert zurück, indem Sie auf das "Ausgefüllter Trichter"-Symbol klicken.

Wenn ein Datenabfragefilter für eine Tabelle erforderlich ist, aber keiner gesetzt wurde, werden die Filter, die als Pflicht-Filter dienen können, mit einem roten Stern "*" markiert. Wählen und konfigurieren Sie einen dieser Filter, um Daten vom Server zu laden. Solange der Tabelle ein Pflicht-Filter fehlt, bleibt der Datenbereich leer und es wird eine Fehlermeldung angezeigt.

In einigen Fällen kann es trotzdem zu vielen Zeilen geben, um sie vom Server abzurufen. In diesem Fall wird eine Fehlermeldung angezeigt, die Sie auffordert, weitere Filter zu setzen. Ändern Sie entweder die Kriterien der vorhandenen Datenabfragefilter oder fügen Sie einen weiteren Datenabfragefilter hinzu.

Spaltenschnellfilter

Spaltenschnellfilter werden auf die Daten angewendet, nachdem sie vom cadisSERVER abgerufen wurden. Wenn ein Schnellfilter geändert wird, werden die Daten nicht erneut vom Server geladen.

Alle Spalten, die in einer Tabelle angezeigt werden, können in einem Schnellfilter verwendet werden. Lesen Sie diesen Artikel über cadisSTUDIO-Tabellenfunktionen, um zu erfahren, wie Sie diese verwenden: https://cadissoftware.atlassian.net/wiki/x/wQC_Ug

Der Tabellenfuß in cadisSTUDIO zeigt immer eine Zeilenanzahl in der Form "Zeilen: XX von YY" an. Die Zahl "XX" zeigt, wie viele Zeilen mit dem angewendeten Schnellfilter tatsächlich angezeigt werden. Die Zahl "YY" zeigt, wie viele Zeilen angezeigt würden, wenn kein Schnellfilter gesetzt wäre. "YY" basiert auf dem Ergebnis, das durch den Datenabfragefilter zurückgegeben wurde.
