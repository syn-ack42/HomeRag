CADIS Knowledge Base : Trip relevant instructions

Sometimes it is needed to know if specific instructions exist on a given trip before the trip starts. With reference to the “locker delivery” use case and having classic locks (no PIN codes) it is clear that the driver needs to take the correct key(s) into his truck before starting the trip. Therefore it is now possible to define instructions that are relevant for the trip.

Preparation

To define a new instruction with the property “trip relevant” you have two possibilities:

Import via masterdata

Set the property “isTripRelevant” in the corresponding master data .xml file and import the file.

<HandlingInstruction>
    <HandlingInstructionData>
        <Code>33</Code>
        <Text>
            <bct:LanguageString LanguageCode="en" CountryCode="EN" isDefault="true">Key: %s</bct:LanguageString>
            <bct:LanguageString LanguageCode="de" CountryCode="DE" isDefault="false">Schlüssel: %s</bct:LanguageString>
        </Text>
        <isTripRelevant>1</isTripRelevant>
    </HandlingInstructionData>
</HandlingInstruction>

Create via cadisSTUDIO

Steps to create an “Instruction”:

  1. Open cadisSTUDIO

  2. Go to “Master data”

  3. Select “Instructions”

  4. Click “Create” button

  5. Enter “ID”, “Text” and select the property “Trip relevant”

  6. Click “Save” button

The “Instruction” is now available to be used in cadis.

Show instructions

In cadisSTUDIO

Precondition: A trip with a few shipments has been imported. The instructions have been referenced to the shipment by the import.

To show the instructions for a trip follow the following steps:

  1. Open cadisSTUDIO

  2. Go to “First & last mile”

  3. Select the trip

  4. Click on the header (vehicle) to open the trip details

  5. Select the tab “Instructions”

In cadisAPP

  1. Open the cadisAPP

  2. Go to “Loading”

  3. Select the loading list

  4. Open the context menu for the loading list

  5. Select “Show trip details”

  6. Select the tab “Instructions”
