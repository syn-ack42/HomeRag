CADIS Knowledge Base : How to allocate an expected storing location

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

The user has the possibility to allocate a storing position to a shipment.

Permissions

The permission "editItemPlannedStoringPosition" is required.

The permission is assigned to the roles "Administrator", "LT-Dispatcher", "Area Freight Forwarder" and "TerminalManager" by default.

Process in cadisCONTROL

  • Select one or more shipments in the shipment overview.

  • Choose “Allocate expected storing position” in the context menu.

A new mask opens.

The "Terminal-Location-ID" – "Terminal-Location-Description" can be selected from masterdata in the selection field. After saving the information appears in the shipment overview under "Planned storing position".

Process in cadisSTUDIO

The function to allocate an expected storing location can be found in the "Ordermanagement" – "Orders".

  • Select one or more shipments.

  • Click on “Allocate expected storing location”.

A new mask opens.

After saving the information appears in the shipment overview under "Planned storing position".
