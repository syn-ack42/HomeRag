CADIS Knowledge Base : Display and edit gates for loading

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Gates can be recorded for a loading. These are visible in “My loadings” for a loading list in the respective details and can also be edited depending on the status of the loading list.

View gates

The gates for a loading list can be shown in the “Gates” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Gates”

Add gates

The gates for a loading list can be added in the “Gates” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Gates”

  4. Click “Edit” button

  5. Modify gate

  6. Click “Save” button

Edit gates

The gates for a loading list can be edited in the “Gates” widget.

  1. Open “My loadings”

  2. Select a loading list

  3. View “Gates”

  4. Click “Edit” button

  5. Modify gate

  6. Click “Save” button
