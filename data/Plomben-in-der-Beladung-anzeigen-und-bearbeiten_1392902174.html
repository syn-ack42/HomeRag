CADIS Knowledge Base : Plomben in der Beladung anzeigen und bearbeiten

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Zu einer Verladung können Plomben erfasst werden. Diese sind unter “myLoadings” für eine Ladeliste in den Details sichtbar und können je nach Status der Ladeliste auch bearbeitet werden.

Plomben ansehen

Die Plomben für eine Ladeliste können im Widget “Plomben ” angezeigt werden.

  1. Öffnen Sie “Meine Ladungen”.

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Plomben”

Plomben hinzufügen

Die Plomben für eine Ladeliste können über das Widget “Plomben” hinzugefügt werden.

  1. Öffnen Sie “Meine Ladungen”.

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Plomben”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Plomben modifizieren

  6. Klicken Sie auf die Schaltfläche “Speichern”.

Plomben bearbeiten

Die Plomben für eine Ladeliste können im Widget “Plomben” bearbeitet werden.

  1. Öffnen Sie “Meine Ladungen”.

  2. Wählen Sie eine Ladeliste

  3. Ansicht “Plomben”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Plombe modifizieren

  6. Klicken Sie auf die Schaltfläche “Speichern”.
