CADIS Knowledge Base : Handle warnings on completion of a loading

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

After the loading process is complete, several checks are carried out, such as loading locks, food transportation, temperature ranges, and so on. Most of these checks are already performed during trip planning (dispatching). However, it is possible that properties are changed during shipment updates by the host system, which may prevent loading completion or require the user to perform additional checks.

There are two types of checks: those that can be overridden and those that prevent completion (blocking checks).

Overridable checks

Loading can be completed even if all checks can be overridden and warnings are present.

You have two options in this example:

  • Use Trip Planning (MyTrips) to cancel the affected shipment

  • Release the loading lock for shipment 265

Blocking checks

Loading cannot be completed if a check is blocked, even if there are warnings.

You have two options in this case:

  • Use Trip Planning (MyTrips) to cancel the affected shipment

  • Update loading restrictions via Host Update
