CADIS Knowledge Base : Lademittel einer Beladung anzeigen und bearbeiten

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Zu einer Beladung können Lademittel erfasst werden. Diese sind unter “Meine Beladungen” zu einer Ladeliste in den jeweiligen Details sichtbar und können je nach Status der Ladeliste auch bearbeitet werden.

Lademittel anzeigen

Die Lademittel einer Ladeliste können im Widget “Lademittel” angezeigt werden.

  1. Öffnen Sie “Meine Beladungen”

  2. Wählen Sie eine Ladeliste

  3. Anzeige der “Lademittel”

Lademittel hinzufügen

Die Lademittel für eine Ladeliste können im Widget “Lademittel” hinzugefügt werden.

  1. Öffnen Sie “Meine Beladungen”

  2. Wählen Sie eine Ladeliste

  3. Gehen Sie zum Widget “Lademittel”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Lademittel ändern

  6. Schaltfläche “Speichern” anklicken

Lademittel bearbeiten

Die Lademittel für eine Ladeliste können im Widget “Lademittel” bearbeitet werden.

  1. Öffnen Sie “Meine Beladungen”.

  2. Wählen Sie eine Ladeliste

  3. Gehen Sie zum Widget “Lademittel”

  4. Schaltfläche “Bearbeiten” anklicken

  5. Lademittel ändern

  6. Schaltfläche “Speichern” anklicken
