CADIS Knowledge Base : Messages

Created by Julian Beyer, last modified on Aug 21, 2025

You reach the Messages are in cadisSTUDIO via the tile

The overview shows all available chats on the left. After selecting a chat, an already started chat can be continued directly in the overview:

The dialogues can be filtered by date.

Using the button Create starts a new communication. You can choose between vehicle and vehicle group.

After selecting, a dialogue opens. In this dialogue the message itself is entered and the vehicle respectively the vehicle group (Message is sent to all vehicles in the selected vehicle group) is selected via checkbox.

The input is confirmed and the chat started via the button Send.

Alternatively a chat can be started with a trip (Precondition: Trip is released): Area Cross dock -> My loadings -> select a trip -> Write message
