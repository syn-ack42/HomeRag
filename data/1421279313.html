Vous accédez à la fenêtre « Liste des expéditions » en cliquant sur le symbole :sendungsuebersicht: dans la barre des tâches cadis.

La fenêtre « Liste des expéditions » affiche toutes les expéditions qui correspondent aux critères de filtrage sélectionnés dans la partie supérieure de l’affichage. La liste est mise à jour automatiquement toutes les 30 secondes, mais elle peut également être mise à jour manuellement en cliquant sur le symbole :refresh:.

Filtres

Pour disposer d’un tableau présentant le maximum d’expéditions que possible, vous pouvez masquer la zone de filtrage en utilisant le bouton :FilterUp: et l’afficher à nouveau en utilisant le bouton :FilterDown:.

Différents filtres sont présents par défaut, vous pouvez les sélectionner dans la liste déroulante Filtres (1ère ligne). Les critères de filtrage proposés seront différents selon le filtre sélectionné

Vous pouvez filtrer les expéditions en fonction de différents critères.

Chaque filtre contient plusieurs champs (critères) qui peuvent être utilisés seuls ou combinés pour le filtrage. Certains critères sont obligatoires (marqués d’un fond jaune).

Chaque filtre contient plusieurs champs (critères) qui peuvent être combinés pour affiner la recherche, certains d’entre eux sont obligatoires (marqués d’un fond jaune) d’autres facultatifs.

Les critères utilisés sont simples – liés au filtre sélectionné en première ligne – mais correspondent aux recherches classiques, quelques exemples de critères :

  • Date d’exécution
    « Aujourd’hui » signifie que la date d’exécution des opérations est celle du jour.
    Vous pouvez modifier cette date en sélectionnant la valeur souhaitée dans la liste (choix sur liste).

  • « Import manqué de l’expédition »
    Si les expéditions sont importées à partir d’un système hôte (application tierce de type ERP, EDI etc…), une mise à jour peut être rejetée par cadis, si l’expédition a déjà été traitée dans cadis, par exemple. L’information est notée sur l’expédition comme un événement nécessitant une confirmation et peut être acquittée via le menu contextuel (voir ci-dessous), ce filtre est utilisé pour filtrer les expéditions correspondantes.

  • « Masquer expéditions principales »
    Si une expédition n’est pas entièrement livrée, le reste de l’expédition sera livré lors d’une livraison ultérieure. Ce filtre permet de n’afficher que la livraison à venir, l’ensemble des expéditions présentes dans la livraison principale ne sont pas affichées.
    Cela correspond à une livraison partielle (manquant ou oubli).

  • Filtre par caractéristiques d’expédition (icones)
    Vous pouvez filtrer en utilisant les critères suivants pour affiner votre recherche:
    :reglerMitteGrau: Caractéristiques n’est pas prise en considération lors du processus de filtrage
    :reglerObenGruen: Les expéditions qui ont cette caractéristique sont filtrées
    :reglerUntenRot: Les expéditions qui n’ont pas cette caractéristique sont filtrées

  • Sélection multiple
    Pour certains filtres dont les valeurs de filtrage sont sélectionnées à partir de listes de sélection, la sélection multiple est également possible:
    par exemple : Le type de transport peut être sélectionné dans la liste de sélection par sélection unique

    ou en cliquant sur le symbole :AbweichFilter: vous pouvez sélectionner plusieurs valeurs:

  • Données annulées
    Filtre supplémentaire placé au-dessus de la zone Filtres. Si cette case est cochée, les expéditions supprimées sont affichées, elles apparaissent sur un fond noir dans la colonne « Supprimer »

Tableau

Les textes et symboles affichés dans les entêtes des colonnes précisent le contenu de ces colonnes ; des info-bulles plus détaillées apparaissent lorsque vous survolez ces entêtes avec votre souris.

Menu contextuel

Le menu contextuel (clic droit dans le tableau) vous permet d’effectuer diverses actions sur les expéditions sélectionnées. Les actions qui sont affichées en gris ne peuvent pas être sélectionnées pour les expéditions sélectionnées, la raison de la non disponibilité apparait sous forme d’une info-bulle si vous laissez le pointeur de la souris survoler l’action.

Actions (proposées par le menu contextuel)

Affecter (Tournée), Affecter (Traction):

Ouvre la fenêtre « Affectation », dans laquelle vous pouvez sélectionner la tournée / traction à laquelle les expéditions sélectionnées seront affectées une fois la validation effectuée (appui sur le bouton affecter).

Détails des expéditions

Ouvre la fenêtre « Détails » de l’expédition.
Si vous sélectionnez plusieurs expéditions, vous pouvez faire défiler les expéditions sélectionnées dans l’écran Détails en appuyant sur les boutons « Expédition précédente » / « Expédition suivante ».

Affectation automatique

Déclenche la fonction d’affectation automatique pour les expéditions sélectionnées, la destination du traitement ayant déjà été déterminée au préalable par le système d’affectation par le code postal par la définition de règles d’affectation automatique.

Disqualifier

Permet d’annuler les enlèvements définis dans cadisCONTROL.

Démarrer l’optimisation des tournées avec les éléments sélectionnés

Ouvre la fenêtre d’optimisation des tournées dans laquelle vous entrez les paramètres d’optimisation. L’optimisation des tournées se lancera automatiquement et la fenêtre « Gestion des optimisations » s’ouvrira.

Liste de chargement, Liste de déchargement, Statut d‘inventaire

Ouvre la fenêtre « Détails du chargement », « Détails du déchargement », « Inventaire ».

Ordre correspondant

Ouvre la fenêtre « Détails de l’ordre » s’il a été mis à la disposition du système mobile après le début de la tournée.

Tournée correspondante

Si l’expédition fait partie d’une tournée, la fenêtre « Liste des tournées » (Tourmonitor) s’ouvre ; la tournée et l’arrêt correspondants sont sélectionnés.

Affectation autorisée

Vous permet d’apposer à l’expédition le statut Prêt pour affectation. Si vous souhaitez livrer une expédition après un problème à la livraison dans une seconde tournée, vous devez apposer le statut Prêt pour affectation afin que cette expédition apparaisse dans la liste des expéditions disponibles et puisse être réaffectée soit manuellement soit à l’aide de la fonction d’affectation automatique.

Saisir le statut d’enlèvement

Ouvre la fenêtre « Modifier le statut » de l’enlèvement et vous permet de définir le statut d’enlèvement en lieu et place du conducteur du véhicule de tournée ou si vous souhaitez corriger le statut attribué par le conducteur.

Apposer un statut de traitement

Ouvre la fenêtre « Saisir le statut de traitement » qui permet de définir un statut de traitement pour enregistrer les étapes de traitement qui ont lieu en dehors de cadis, par exemple, les processus de clarification des anomalies de livraison.

Vous pouvez sélectionner un statut et saisir un texte explicatif dans cet écran.

Le statut de traitement courant est affiché dans la liste des expéditions, dans la colonne « Stst Expédition » / statut de traitement de type expédition, une info-bulle apparait et affiche le statut courant, le texte entré manuellement et le nom de l’utilisateur qui a saisi le statut.

Modifier la date d’exécution

Ouvre la fenêtre « Modifier la date d’exécution » qui vous permet de modifier les livraisons ou enlèvements d’une expédition à une autre date. L’expédition n’apparaîtra alors plus dans les expéditions programmées pour la première date et ne sera donc pas pris en compte par la fonction d’affectation automatique pour cette journée, elle apparaitra à la date nouvellement sélectionnée.

Modifier les instructions de chargement

Ouvre la fenêtre « Détails » de l’expédition et donne la possibilité de modifier ou de supprimer des textes d’information (instructions) existant ou d’en saisir de nouveaux

Modifier le groupe d’expéditions

Ouvre la fenêtre « Modifier le groupe d’optimisation » qui vous permet soit d’affecter l’expédition à un groupe d’optimisation soit de modifier l’affectation du groupe d’expéditions qui a été effectuée par les règles d’affectation.
Vous sélectionnez un groupe dans la liste des groupes existants pour le modifier ou l’affecter.

Blocage de l’affectation / du chargement, Retirer le blocage de l’affectation / du chargement

Vous utilisez cette fonction pour régler ou lever manuellement une interdiction de chargement qui a été définie dans cadis. Vous ne pouvez pas lever dans cadisCONTROL une interdiction de chargement qui a été définie par un système hôte. Les interdictions de chargement sont indiquées par le symbole de cadenas:schloss: dans la colonne :verladesperre:, l’info-bulle indiquant le motif du blocage.

Suppression de l’enlèvement

Vous pouvez supprimer les enlèvements qui ont été créés dans cadis.

Dégrouper les palettes de groupage

Ce menu apparaît pour les palettes de groupage (palettes master) uniquement: la palette de groupage est déconsolidée, les UM de cette palette ne sont plus attribuées à une palette master.

Clôturer la palette de groupage

Ce menu apparait pour les palettes de groupage (palettes master) uniquement : la palette de groupage est clôturée, aucune poursuite de traitement n’est possible (ajout, suppression d’UM).

Confirmer l’import manqué de l’expédition

Confirme un import manqué de l’expédition par cadis (voir descriptif du filtre Import manqué de l’expédition)

Refuser

Vous pouvez refuser les enlèvements des autres agences.

Modifier la priorité

Vous pouvez attribuer une nouvelle priorité à l’expédition en saisissant un numéro ici.

Imprimer

Imprimer l’annexe matières dangereuses, le récépissé de livraison, l‘enlèvement,
le rapport de température
