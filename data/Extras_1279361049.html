You reach the Extras screen using the  symbol in the cadis control panel.

This menu contains different functions for administration and settings.

Save Settings

Use to save the current settings for all open windows (size, position, displayed columns, their size and order) under a name under which you can use to restore these settings at any time.

Use to delete the current settings.

User Preferences

Personal settings. The settings options are explained in the screen.

Change Log On

Allows a new login to cadis with a different user, to another depot or using a different language without leaving cadis and restarting.

Change Password

User Management

Create, delete and change users and roles

This screen shows the list of users.

To create or change a new user, the User screen is displayed:

If Password must be changed by user is set, the user must change the password at the first login.

You assign one or more roles to each user.

License Management

Opens the license manager in the section "Administration" in cadisSTUDIO.

Generate recurring collection orders

You can create regular collection orders for several days in advance.

Mobile devices

The screen shows an overview of the mobile systems used.

You see the time of the last registration and the vehicle for which the mobile device is currently registered.

You can request a log file from a mobile device by selecting an entry. You can see the request in the right section of the window. There you can see when the log file has been transferred and you have access to the log files.

The mobile device ID is not the serial number of the device, but a random number generated by the app.

Mobile Accesses

All mobile releases that you have issued are entered here.

You issue new mobile releases for drivers who are permanently assigned to a vehicle in the List of vehicles.

New mobile releases for one or several vehicles, for a scanner or a driver can be issued here (New button).

You can edit or revoke all mobile shares by selecting the corresponding entry.

Mobile software versions

Manages the software versions for the mobile devices.

The Create button is used to add a new software package onto the server. In this dialog you select the zip file with the mobile software package.

You can then assign the uploaded software versions as follows:

  • To all mobile devices in all depots

  • To all mobile devices in a depot

  • To individual vehicles or terminal scanners

The corresponding mobile devices will then inform you that a new software version is available for installation when you log on. As long as the version is not set as mandatory, the employee can postpone the installation.
