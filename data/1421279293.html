Sie erreichen die Sendungsübersicht über das Icon :sendungsuebersicht: in der cadis-Bedienleiste:

Die Sendungsübersicht zeigt alle Sendungen an, die den gewählten Filterkriterien entsprechen. Die Übersicht wird alle 30 Sekunden automatisch aktualisiert, kann aber zusätzlich auch über das Symbol :refresh: manuell aktualisiert werden.

Der Filterbereich

Um möglichst viele Sendung anzuzeigen, können Sie den Filterbereich mit :FilterUp: ausblenden und mit :FilterDown: wieder einblenden.

Sie können die Sendungen nach verschiedensten Gesichtspunkten filtern. Für unterschiedliche Anwendungsfälle sind verschiedene Filtersätze implementiert, die Sie über die Listbox Filter auswählen.

Jedes Filterset enthält unterschiedliche Felder, nach denen gefiltert werden kann. Ein oder mehrere Felder sind jeweils die Hauptfilterkriterien. Diese müssen immer angegeben werden, die Felder sind mit gelbem Hintergrund als Pflichtfeld gekennzeichnet.

Die meisten Filtermöglichkeiten sind selbsterklärend. Hier einige Besonderheiten:

  • „heute“ im Namen des Filtersets bedeutet, dass das Ausführungsdatum mit dem heutigen Datum vorbelegt ist. Sie können das aber beliebig ändern.

  • Gescheiterter Sendungsimport: Werden Sendungen aus einem führenden System importiert, kann es passieren, dass ein Update von cadis abgelehnt wird, wenn die Sendung in cadis z.B. bereits final bearbeitet wurde. Dies wird an der Sendung als quittierungspflichtiges Ereignis vermerkt und über das Kontextmenü quittiert werden (siehe unten). Mit diesem Filter werden entsprechende Sendungen gefiltert.

  • Hauptsendungen ausblenden: Wenn eine Sendung nicht vollständig zugestellt wurde, wird die Restsendung über eine Nachliefersendung ausgeliefert. Dieser Filter sorgt dafür, dass in solchen Fällen nur die Nachliefersendung aber nicht die Hauptsendung gefiltert wird.

  • Filtern nach Sendungseigenschaften: für viele Sendungseigenschaften kann folgendermaßen gefiltert werden:
    :reglerMitteGrau: Eigenschaft wird beim Filtern nicht berücksichtigt
    :reglerObenGruen: Sendungen, die diese Eigenschaft haben, werden gefiltert
    :reglerUntenRot: Sendungen, die diese Eigenschaft nicht haben, werden gefiltert

  • Mehrfachauswahl
    Für einige Filter, deren Filterwerte aus Auswahllisten gewählt wird, ist auch eine Mehrfachauswahl möglich:
    z.B.: Verkehrsart kann per Einfachauswahl aus der Auswahlliste gewählt werden

    oder über den Button :AbweichFilter: können mehrere Werte gewählt werden:

  • Stornierte Werte: Dies ist ein zusätzlicher Filter oben in der Icon-Zeile. Ist diese Checkbox angeklickt, dann werden zusätzlich stornierte Sendungen angezeigt, die in der Spalte „Storniert“ gekennzeichnet sind

Die Tabelle

Die Spalten sind selbsterklärend. Bei unklaren Fällen, lassen Sie sich die Bedeutung der Spalte als Tooltip anzeigen, indem Sie mit der Maus auf dem Spaltenkopf kurz verweilen. Bei unklarem Feldinhalt können Sie auch hier in vielen Fällen über den Tooltip weitere Details erhalten.

Das Kontextmenü

Über das Kontextmenü (Drücken der rechten Maustaste auf der Tabelle) können Sie diverse Aktionen an den selektierten Sendungen durchführen. Aktionen, die grau dargestellt sind, können Sie für die gewählten Sendungen nicht anwählen. Den Grund dafür sehen Sie als Tooltip, wenn Sie den Mauszeiger eine kurze Zeit auf dem Eintrag ruhen lassen.

Aktionen

Disponieren (Nahverkehr) / (Fernverkehr):

Sie werden in die Maske Disposition geleitet, in der Sie eine Nahverkehrstour / Fernkehrsabfahrt auswählen, auf die die selektierten Sendungen disponiert werden.

Sendungsdetails

Zeigt die Sendungsdetails an. Bei mehreren selektierten Sendungen können Sie in der Maske Sendungsdetails durch die selektierten Sendungen blättern.

Ankunftsbenachrichtigung

Ist nur verfügbar für NV-Sendungen / Abholungen, die die entsprechende Eigenschaft aufweisen. Hierüber wird die Benachrichtigung ausgelöst (wenn konfiguriert), zumindest aber dokumentiert.

Gebietsspediteur

Hierüber kann die Sendung einem Gebietsspediteur / Masterdisponent zugewiesen werden bzw. wieder zurückgesetzt werden.

Autodispo

Hierüber lösen Sie die Autodispo-Funktion für die selektierten Sendungen aus, wobei das Dispositions-Ziel bereits zuvor über PLZ-Routing und das Regelwerk Autodispo bestimmt wurde.

Aussteuern

Hiermit setzen Sie die Sendung manuell auf ausgegangen, Als Ladeliste wird Aussteuerung eingetragen, der Hallenstatus ist nicht auf Halle und der Dispostatus ist nicht disponierbar.

Freie Tourenoptimierung mit Auswahl starten

Hierüber starten Sie mit den selektierten Sendungen die freie Tourenoptimierung. Sie werden in Maske Tourenoptimierung geleitet, in der Sie die Optionen für die Optimierung definieren.

Ladeliste, Entladeliste, Inventurliste

Öffnet die entsprechende Liste, falls die Sendung auf einer solchen Liste war.

Zugehöriger Auftrag

Zeigt den Abhol- bzw. Zustellauftrag an, wenn er nach Tourfreigabe der Nahverkehrstour für das Mobilsystem bereitgestellt wurde.

Zugehörige Tour

Wenn sich die Sendung auf einer Nahverkehrstour befindet wird der Tourmonitor geöffnet und darin die zugehörige Tour und der zugehörige Stopp selektiert.

Sendung disponierbar

Hierüber setzen Sie die Sendung in den Status disponierbar. Sendungen, die sie z.B. nach einem Zustellhindernis noch einmal zustellen wollen, müssen Sie wieder auf disponierbar setzen, damit sie wieder in der Liste der disponierbaren Sendungen erscheinen und manuell oder über die Autodispo-Funktion disponiert werden können.

Nachliefersendung anlegen

Eine Nachliefersendung (Teilsendung) wird für die Sendung erzeugt.

PUD-Status setzen

Hierüber setzen Sie den Abhol- bzw. Zustellstatus einer Nahverkehrssendung, wenn dieser nicht durch den Fahrer des Nahverkehrsfahrzeugs gesetzt wurde oder wenn Sie den vom Fahrer vergebenen Status korrigieren wollen.

Bearbeitungsstatus setzen

Sie können einen Bearbeitungsstatus setzen damit Bearbeitungsschritte festhalten, die außerhalb von cadis stattfinden, z.B. Klärungsprozesse bei Zustellhindernissen.

Dazu können Sie in dieser Maske einen Status und einen erläuternden Text erfassen.

Der aktuelle Bearbeitungsstatus wird in der Sendungsübersicht in Spalte Bsts Sendung angezeigt, als Tooltip komplett mit Text und Name des Benutzers, der den Status erfasst hat.

Ausführungsdatum ändern

Wenn Sie eine Sendung an einem späteren Tag zustellen oder abholen wollen, dann können Sie das Ausführungsdatum ändern. Die Sendung erscheint dann nicht mehr in den disponierbaren Sendungen für den aktuellen Tag und wird somit auch von der Autodispo-Funktion für den aktuellen Tag nicht berücksichtigt.

Hinweistext bearbeiten

Die Maske Sendungsdetails wird geöffnet mit der Möglichkeit Hinweistexte zu ändern, zu löschen oder neue Hinweistexte zu erfassen

Sendungsgruppe ändern

Mit dieser Funktion können Sie die Gruppenzuordnung der Sendung ändern, die über das Regelwerk Sendungsgruppenermittlung erfolgte und die Sendung einer anderen Gruppe zuweisen.
Sie wählen dazu aus der Liste der bereits existierenden Gruppen eine aus:

Dispo/Verladesperre setzen / aufheben

Hierüber setzen Sie manuell eine Verladesperre oder heben eine Verladesperre, die in cadis gesetzt wurde wieder auf. Verladesperren, die das Hostsystem gesetzt hat, können Sie in cadis nicht aufheben. Verladesperren werden in Spalte :verladesperre: mit einem Schloss-Symbol :schloss: gekennzeichnet, Der Tooltip gibt den Grund der Verladesperre an.

Abholer stornieren

Abholungen, die in cadis erfasst wurden, können Sie hierüber stornieren.

Masterpalette auflösen

Nur für Masterpaletten: Die Masterpalette wird aufgelöst. Die bisher enthaltenen Packstücke sind jetzt keiner Masterpalette mehr zugeordnet.

Masterpalette schließen

Nur für Masterpaletten: Masterpalette wird geschlossen, weitere Bearbeitung (Hinzufügen, Entfernen von Packstücken) ist nicht mehr möglich.

Gescheiterten Sendungsimport quittieren

Quittieren eines von cadis nicht akzeptierten Sendungsimports. (siehe oben Filter gescheiterter Sendungsimport)

Ablehnen

Ablehnen von Abholungen, die andere Depots eingestellt haben.

Priorität ändern

Vergeben Sie eine neue Priorität für die Sendung indem Sie hier eine Zahl eingeben.

Drucken

Gefahrgutanhang, Lieferschein, Abholauftrag, Temperaturbericht, Details mit/ohne Notizen drucken.

Öffnet das entsprechende Dokument im PDF-Reader. Von dort können sie es ausdrucken oder speichern.
