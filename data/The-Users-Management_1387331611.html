CADIS Knowledge Base : The Users Management

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

You can create your users & associate profiles that you define by access rights in cadis.

Principles

① Create user profiles. By associating the rights available in cadis, you can conceive hierarchical profiles

② Create your users. ID + password for users expected to use one of CADIS tools

③ Assign profiles to users. This is where users get the rights you have distributed to their profiles in step ①

④ Grant access to the network. You can select agencies from your network to which your users can get access to

⑤ Allow a user management level. 3 Management levels : users globally/ users locally/none

How to Create or modify the user profiles

① Open
the « Users
& Devices» menu. In the « User Roles » tab, click on « Create » or double click on a role to see it’s details

② Open the drop down list in the Permissions field, where you can select to add (or remove) the desired right

③ Save
your changes

How to Create, Edit or Delete a user

① Open the « Users & Devices» menu. In the User Management tab, click on « Create » or double-click on a User to edit

② Fill in the mandatory fields

③ System: user can manage the whole cadis system (do everything without restrictions & without any roles assigned

Users globally: user can create/edit users in all depots

Users locally: user can create/edit users in his depot only

Single Sign-on with cadis

User management can be delegated to your host system as cadis login accepts openID connect. Get in touch with our team for further detail.
