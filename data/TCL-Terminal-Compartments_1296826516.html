CADIS Knowledge Base : TCL Terminal Compartments

Created by Julian Beyer, last modified on Aug 19, 2025

Terminal Compartments are a necessary precondition for cadis to monitor packages in the terminal.

A Terminal Compartment is an area in a terminal

  • that is temperature controlled: The temperature is thus supposed to be within a certain temperature range

  • which has a relevant temperature sensor defining its temperature over time

  • with one or usually several terminal locations located therein: Goods stored at such terminal locations that are within a terminal compartment and can be temperature controlled by the relevant sensor.

You can read the Terminal Compartments in all depots.
Select the Frame “Terminal Locations” in cadisTCL:

You can create/edit Terminal Compartments in your own depot by clicking “Create”:

  • Compartment:

Choose a name which identifies your compartment. This name is used to identify the compartment whenever a package is stored there, and thus used whenever you create the temperature report for a shipment. You can change this name and still receive proper temperature reports.

  • Relevant Sensor:
    Choose the Relevant Sensor from a dropdown list: You find all sensors that sent data to cadis via he configured telematics provider interfaces.
    Sensors that have already been defined as relevant sensor of another compartment are deactivated in this list: One sensor can be relevant sensor of at most one compartment.

  • ThermoRange:

Choose the ThermoRange from a dropdown list: You can choose the ThermoRanges from a dropdown list: Alternatives are 2-8°C and 15-25°C.
