CADIS Knowledge Base : How to Use the Advanced Table Features in cadisSTUDIO

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Tables in cadisSTUDIO provide many advanced features (as of R4.21).

Sorting By Column

To sort by a column, click on the column header. Click again to reverse the sort order, and a third time to remove sorting altogether. Hold the "CTRL" key while clicking on column headers to sort by multiple columns. An up or down arrow in the column will indicate the sort order. If you sort by multiple columns, a small number will indicate the sort priority ("1" means sorting by this column first, then "2," and so on).

Column Management Tab

Click on the "Columns" control on the right-hand side of each table in cadisSTUDIO to open a menu that allows you to conveniently show/hide columns and sort them via drag & drop (of course you can also chnage the column order by dragging and dropping the column headers int he able directly). You can use the "Search" box to find a specific column in the list. Drag and drop a column to the bottom to use the "Group by" feature.

Quick Column Filter Tab

Click on the "Columns" control on the right-hand side of each table in cadisSTUDIO to open a menu that allows you to easily manage quick column filters. When a quick filter is applied, it will appear in a small bubble with a white background above the table. Click the small filter symbol in the bubble to remove the filter.

Group by Column

Use either the Column Management Tab or the column menu to group a table by one or more columns.

Individual Column Menu

Many advanced features are accessible via the "3 dot" button on the right edge of each column, which opens the column menu. In the menu, use the "Funnel" symbol to set quick filters and the "3 bars" tab to access advanced column features. Here, you can:

  • Pin the column to the left or right edge of the screen, so it remains visible when you scroll the table horizontally.

  • Automatically adjust the width of this column or all columns to match the content.

  • Group by the current column.

  • Reset the columns.
