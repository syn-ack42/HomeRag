CADIS Knowledge Base : Can I work on several loading lists / unloading lists at the same time?

Created by Mohamed.Shebl, last modified by Julian Beyer on Sep 08, 2025

Yes! You can process several lists at the same time (if you have loaded these lists on your mobile device). If you are scanning a package, that does not belong to the current loading list, but to another one that you are working on, you can switch to this list. The package is then documented as loaded/unloaded.
