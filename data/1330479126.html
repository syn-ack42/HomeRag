CADIS Knowledge Base : Which Reference Number is printed where on the ePOD?

Created by Julian Beyer, last modified on Sep 08, 2025

A common question is where each Reference Number is printed on a ePOD. We are assuming ctDepotItem is used for shipment data import.

  • the OrderNumber and GlobalTrackingNumber refer to the the correspondingly named elements in the cTExpectedDepotItem xml

  • if OrderNumber is empty (which is allowed) the HostID of the shipment is used instead.

  • the Shipper-/Consignor-/ConsigneeReferenceNumber refer to a “References” entry with that ReferenceType
