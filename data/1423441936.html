CADIS Knowledge Base : Caisses mobiles / conteneurs et Types de Caisses mobiles / Conteneurs

Created by Julian Beyer on Aug 21, 2025

Vous accédez aux fenêtres « Caisses mobiles/Conteneurs » et « Caisses mobiles/ Types de conteneurs » à partir de la fenêtre « Référentiels » (:regeldefinition: dans la barre des tâches cadis) : puis cliquez sur les onglets « Caisses mobiles/Conteneurs » et « Caisses mobiles/ Types de conteneurs » respectivement.

A partir de la page « Référentiels » de cadisSTUDIO vous sélectionnez dans la colonne de gauche « Caisses mobiles / conteneurs » et « Types de caisses mobiles / conteneurs » respectivement.

Vous pouvez alors, créer, modifier ou supprimer une Caisses mobile / conteneur ou un Type de caisses mobiles / conteneurs resp. en le sélectionnant (création/modification/suppression accessible selon vos droits utilisateurs et le type de données centrales ou locales).

Dans cadis, un conteneur ou une caisse mobile est défini comme tout élément mobile qui a un espace de chargement mais aucune roue, c’est-à-dire qu’un conteneur ou qu’une caisse mobile ne sont pas des véhicules au sens cadis.

Les types de conteneurs et types de caisses mobiles définissent les caractéristiques communes – dimensions, poids, etc… – des conteneurs / caisses mobiles utilisées dans cadis.

Types de Caisses mobiles / conteneurs

Vous saisissez ou modifiez les données décrivant les types de caisses mobiles / conteneurs, seuls les champs marqués d’un astérisque sont obligatoires.

Caisses mobiles / Conteneurs

Vous saisissez ou modifiez les données décrivant chaque caisse mobile / conteneur, seuls les champs marqués d’un astérisque sont obligatoires.

Dès lors que vous utilisez les référentiels du type de caisse mobile / conteneur, les informations associées apparaissent automatiquement et ne sont pas modifiables dans cette page.
